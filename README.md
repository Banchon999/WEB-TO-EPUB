# WebToEpub

ส่วนขยายเบราว์เซอร์ (Browser Extension) สำหรับแปลงหน้าเว็บ — โดยเฉพาะนิยายและเว็บตูน — ให้กลายเป็นไฟล์ **EPUB** หรือ **TXT** เพื่อนำไปอ่านบนเครื่องอ่านอีบุ๊กหรือแอปอ่านหนังสือ

> เวอร์ชัน: `1.0.12.0` · Manifest V3 · รองรับ Chrome เวอร์ชัน 120 ขึ้นไป

---

## คุณสมบัติเด่น (Features)

- 📚 แปลงนิยายหลายตอนจากเว็บไซต์ให้เป็นไฟล์ EPUB เดียว
- 📝 รองรับการบันทึกเป็นไฟล์ TXT
- 🌐 รองรับเว็บไซต์ต้นทางมากกว่า **378 ไซต์** ผ่านระบบ Parser เฉพาะของแต่ละเว็บ
- 🖼️ ดาวน์โหลดและฝังรูปภาพ รวมถึงตั้งค่ารูปหน้าปก (cover image)
- 🔢 เลือกตอนที่ต้องการดาวน์โหลดได้ (chapter selection)
- 🦶 ดึง footnote / เชิงอรรถมารวมในไฟล์
- 🌙 รองรับธีมมืด (Dark mode) ทั้งแบบอัตโนมัติและบังคับเปิด
- 🌍 รองรับหลายภาษา: อังกฤษ (en), รัสเซีย (ru), ยูเครน (uk), เวียดนาม (vi)

## โครงสร้างโปรเจกต์ (Project Structure)

```
WebToEpub/
├── manifest.json          # ไฟล์กำหนดค่าส่วนขยาย (Manifest V3)
├── popup.html             # หน้าต่าง UI หลักของส่วนขยาย
├── book128.png            # ไอคอนส่วนขยาย
├── css/                   # สไตล์ชีต (default, autoDark, alwaysDark)
├── images/                # ไอคอนสถานะตอน (SVG)
├── _locales/              # ไฟล์แปลภาษา (en, ru, uk, vi)
├── @zip.js/               # ไลบรารีสำหรับสร้างไฟล์ ZIP/EPUB
├── dompurify/             # ไลบรารีทำความสะอาด HTML
└── js/
    ├── main.js            # ตัวจัดการหลักของ popup
    ├── Parser.js          # คลาสฐานของ Parser
    ├── ParserFactory.js   # เลือก Parser ตามเว็บไซต์
    ├── EpubPacker.js       # ประกอบไฟล์ EPUB
    ├── TxtPacker.js        # ประกอบไฟล์ TXT
    ├── ImageCollector.js  # ดาวน์โหลดและจัดการรูปภาพ
    ├── Download.js        # จัดการการดาวน์โหลด
    ├── UserPreferences.js # บันทึกการตั้งค่าผู้ใช้
    └── parsers/           # Parser เฉพาะของแต่ละเว็บไซต์ (378+ ไฟล์)
```

## หลักการทำงานโดยย่อ (How It Works)

1. ผู้ใช้เปิดหน้าสารบัญ (Table of Contents) ของนิยายบนเว็บ แล้วกดไอคอนส่วนขยาย
2. `ParserFactory` เลือก Parser ที่เหมาะกับเว็บไซต์นั้น (จากโฟลเดอร์ `js/parsers/`)
3. Parser ดึงรายชื่อตอน เนื้อหา และข้อมูลหนังสือ (metadata)
4. `ImageCollector` ดาวน์โหลดรูปภาพที่เกี่ยวข้อง
5. `EpubPacker` / `TxtPacker` ประกอบทุกอย่างเป็นไฟล์เดียวให้ดาวน์โหลด

---

## การติดตั้ง (Installation — โหมดนักพัฒนา)

1. เปิด Chrome แล้วไปที่ `chrome://extensions`
2. เปิด **Developer mode** (มุมขวาบน)
3. กด **Load unpacked** แล้วเลือกโฟลเดอร์โปรเจกต์นี้
4. ไอคอน WebToEpub จะปรากฏบนแถบเครื่องมือ พร้อมใช้งาน

---

## การเพิ่ม Parser สำหรับเว็บใหม่

แต่ละเว็บไซต์มีไฟล์ Parser ของตัวเองในโฟลเดอร์ `js/parsers/` โดยสืบทอดจากคลาส `Parser`
หากต้องการรองรับเว็บใหม่ ให้สร้างไฟล์ `<ชื่อเว็บ>Parser.js` แล้วลงทะเบียนผ่าน
`parserFactory.register(...)` ตามรูปแบบของ Parser ที่มีอยู่เดิม
